# An interactive shell with CLI inspired by BASH

in order to compile & run on your terminal:
<git clone git@github.com:mekundur/minishell.git>
<make>
<./minishell>


Fetaures:
- pipeline
- redirections
- here document
- error handling
- exit codes
- environment variables
- builtins: echo, pwd, env, export, unset, exit
